## Aadarsh's Tech Blog

This is a simple blog website created by Aadarsh Kumar to document his learning journey in programming and web development. The site is built using HTML and CSS.

### Project Structure

- **index.html**: Home page introducing the blog, author, and topics covered (Python, Data Structures, HTML, Programming Tips).
- **blog.html**: Contains blog posts about programming experiences, especially Python. Each post includes author, date, and content.
- **about.html**: Shares information about the author, his academic journey, and motivation for blogging.
- **contact.html**: Provides a contact form for visitors to send messages (name, email, message).
- **css/style.css**: Styles the website, including navigation bar, headings, and layout. Features hover and active effects for navigation links.

### Features

- Responsive navigation bar linking all pages.
- Clean, readable layout for blog posts and personal information.
- Contact form for user interaction.
- Custom CSS for visual appeal (colored headings, navigation hover/active states).

### How to Use

Open `index.html` in your browser to start exploring the blog. Use the navigation bar to visit other pages.
